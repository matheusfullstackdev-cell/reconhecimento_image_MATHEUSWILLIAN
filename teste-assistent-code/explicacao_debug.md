# Debug de Código

## O que é debug?

Debug é o processo de identificar e corrigir erros em um programa.

---

## Problema encontrado

Divisão por zero causava erro no programa.

Exemplo:
10 / 0 → erro

---

## Solução aplicada

Utilizamos o bloco try/except para tratar o erro:

- try → tenta executar
- except → captura o erro

---

## Resultado

O programa não quebra e mostra uma mensagem amigável.

---

## Conclusão

Debug é essencial para garantir que o sistema funcione corretamente mesmo com erros.
