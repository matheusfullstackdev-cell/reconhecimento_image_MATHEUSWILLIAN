# 🔢 Números Primos

## 📌 O que são números primos?

Um número primo é aquele que possui exatamente dois divisores:
- 1
- Ele mesmo

### Exemplos:
- ✅ 2, 3, 5, 7, 11 → são primos
- ❌ 4, 6, 8, 9 → não são primos

---

## 🧠 Lógica utilizada no código

O algoritmo verifica se um número pode ser dividido por outro número além de 1 e ele mesmo.

Para otimizar:
- A verificação vai até a raiz quadrada do número

---

## ⚙️ Funções criadas

### `eh_primo(numero)`
Verifica se um número é primo.

### `listar_primos(limite)`
Retorna todos os números primos até um valor informado.
