# Refatoração de Código

## O que é refatoração?

Refatoração é o processo de melhorar o código sem alterar seu funcionamento.

---

## Objetivo

- Tornar o código mais limpo
- Melhorar legibilidade
- Facilitar manutenção

---

## O que foi feito

- Uso de funções prontas do Python (sum, max)
- Organização em funções
- Uso de tipagem (list, tuple)

---

## Benefícios

- Código mais curto
- Mais fácil de entender
- Menos chance de erro
