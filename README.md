# 🤖 Reconhecimento de Imagens

Um projeto de reconhecimento de imagens em tempo real usando webcam e Teachable Machine, desenvolvido para demonstrar conceitos de IA aplicada a visão computacional.

## 📋 Descrição

Esta aplicação web permite capturar imagens via webcam e reconhecer objetos ou classes em tempo real usando um modelo de machine learning treinado. É uma demonstração prática de como integrar IA em aplicações web modernas.

## ✨ Funcionalidades

- 🎥 Captura de vídeo em tempo real da webcam
- 🧠 Reconhecimento automático de imagens usando modelo treinado
- 📊 Exibição de probabilidades para cada classe detectada
- 🎨 Interface responsiva e moderna
- ⚡ Processamento rápido com TensorFlow.js

## 🚀 Como executar

1. **Pré-requisitos:**
   - Navegador web moderno (Chrome, Firefox, Edge)
   - Acesso à câmera do dispositivo

2. **Passos:**
   - Abra o arquivo `index.html` em seu navegador
   - Clique no botão "Iniciar Câmera"
   - Permita o acesso à câmera quando solicitado
   - Aponte a câmera para objetos a serem reconhecidos

3. **Modelo:**
   - O modelo treinado deve estar na pasta `my_model/` com os arquivos `model.json` e `metadata.json`

## 🛠️ Tecnologias Utilizadas

- **Frontend:**
  - HTML5
  - CSS3
  - JavaScript (ES6+)

- **IA/ML:**
  - TensorFlow.js
  - Teachable Machine Image
